# webflow-lib

Various utils for Webflow

## Setup

Inside head tag:

    <script defer src="https://cdn.jsdelivr.net/npm/@krujin/webflow-lib@1/dist/index.js" onload="onLibReady()"></script>

Before </body> tag: 

    <script>
        function onLibReady() {
            // your code ...
        };
    </script>

## Usage

Add your code inside the onLibReady() function.
